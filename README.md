# Task 1 - Data Cleaning and Preprocessing

## Objective

The objective of this task is to clean and preprocess a raw dataset using Python and Pandas.

## Dataset Used

Netflix Movies and TV Shows Dataset from Kaggle.

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Steps Performed

1. Loaded the dataset using Pandas.
2. Checked dataset structure and missing values.
3. Removed duplicate records.
4. Handled missing values using fillna() and dropna().
5. Standardized text values and column names.
6. Converted date columns into proper datetime format.
7. Verified and corrected data types.
8. Exported the cleaned dataset.

## Outcome

Successfully cleaned and prepared the dataset for further analysis and visualization.

## Skills Learned

* Data Cleaning
* Data Preprocessing
* Handling Missing Values
* Removing Duplicates
* Data Type Conversion
* Pandas Operations
